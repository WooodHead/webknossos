FAQ

Which browsers are supported for webKnossos?

webKnossos runs best with Google Chrome. We use it to test and develop the software and recommend it.

However, any other modern browser that supports the full WebGl specification should work as well. We have had good success with the following browsers:

  • Firefox 27+
  • Opera 20+
  • Safari 7.0.3 (how to enable WebGl)

At the time of this writing Internet Explorer 11 and below are not supported.
